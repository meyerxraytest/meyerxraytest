    5. XM series (M3) · fishbone[MY8045-XM_Fish_V1.01beta1015.zip](update 2019-10-16)

      6. Explain:
      Xfish is shared by wide and narrow; for fool installation, set the start menu to start automatically, and create a new desktop shortcut key.
      Main update Description:
      1. Dual large screen and single large screen are compatible;
      2. The lighting mechanism of the double large screen is changed. As long as the cache is not full, it will be on, and the single large screen will still be on;
      3. Support screen brightness adjustment in system settings;
      4. Three attempts will be made if communication fails to start;
      5. Added remote assistance function;
      6. Add the function of saving 16bit original drawing in the software parameter interface (loading first);
      7. The default intelligent correction function is used to solve the problem of stripe when starting the machine (hidden in the system setting interface);
      8. The local 16bit or 8bit PNG image can be loaded and displayed automatically.

    8. XL series · large packaging machine[MY12060-XL_BigPACK_V1.00beta1016.zip](update 2019-8-27)

      9. Explain:
      Fool installation, set the start menu to start automatically, and create a new desktop shortcut key.
      Major updates include:
      1. Part of the later installed machine system backplane is 100 lines of data, and the earlier one is 200 lines. The software defaults to 100 lines, which can be switched by changing the name to replace the FMT file in the root directory;
      2. The turnover plate removal is changed from PLC control to upper computer control, so as to solve the problem that cannot be removed continuously. PLC needs to be upgraded to support;
      3. Automatically re detect 3 times when communication detection fails during startup;
      4. Add the words "big package" in the top prompt column;
      5. Ray is turned off by default when testing is started.
